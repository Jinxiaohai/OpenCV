My OpenCV Project
===============

Introduction
------------

OpenCV是一个跨平台的计算机视觉库，其官方的网站为[http://opencv.org](http://opencv.org)。
发布协议遵循BSD，因此可以被自由的用于学术和商业活动中。
其代码主要由C和C++写成，但是为C++，Python和Java等语言都留有接口。
计算机视觉在社会和商业中的流行间接的推动了OpenCV的广泛应用，
这也导致了我对该领域的兴趣，因此OpenCV库的学习也是势在必行。

Features
--------
